# Contributing to AgentKit

Thank you for your interest in contributing!

This repository is currently under active development. To help get started:

1. **Open an issue** describing the feature or bug you want to work on.
2. **Fork the repository** and create a new branch for your changes.
3. **Write tests** for any new functionality or bug fixes.
4. **Follow the existing coding style** and run `ruff`/`black` if configured.
5. Submit a **pull request** referencing the original issue and detailing your changes.

There is no formal process yet for maintaining changelogs or release notes; please add a brief note in `CHANGELOG.md` when merging significant updates.

Thanks for helping make AgentKit better!
